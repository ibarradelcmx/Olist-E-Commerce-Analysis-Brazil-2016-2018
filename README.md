# Olist E-Commerce Analysis
### Data-driven insights on Brazil's largest e-commerce marketplace

## Overview
Analysis of 99,000+ orders from Olist, Brazil's leading e-commerce platform (2016-2018), using SQL window functions, multi-table JOINs, and Looker Studio visualizations.

## Key Findings
- 97% of orders are successfully delivered — exceptional fulfillment rate
- Top seller processes 2,033 orders — nearly double the #10 seller
- Beauty & Health leads all categories with $1.26M in revenue
- Northern Brazilian states (RR, AP, AC) show significantly lower sales activity

## Tools
- MySQL — data storage and SQL analysis
- Looker Studio — dashboard and visualizations
- Data source: Olist Public Dataset (Kaggle)

## Questions Analyzed
1. Order distribution by status
2. Top 10 sellers by number of orders
3. Top sellers by state/region
4. Top 5 product categories by revenue
5. Bottom 5 states by sales volume

## SQL Highlights
- Window functions: RANK(), ROW_NUMBER(), PARTITION BY
- Multi-table JOINs (up to 3 tables)
- Aggregations: SUM, COUNT, ROUND

## Data Source
Olist Brazilian E-Commerce Public Dataset — Kaggle
99,441 orders | 9 tables | 2016-2018
